Ceci est un test Git

    J'ai d'abord créé un dossier sur mon bureau. Puis j'ai ouvert ce dossier dans Visual code. J'ai ensuite tapé mkdir dans le terminal dans Visual code. Il m'a dit qu'un dossier créé git a été créé dans mon ordi. Puis j'ai crée mon index et mon css. Et là miracle : l'icone arborescente de Visual code s'est allumée en signalant 2 changements. j'ai cliqué sur le signe + et ils sont passés en staged je crois. J'ai tapé git status dans le terminal. et il me dit que 2 nouveaux fichiers existent et que les modifications seront enregistrées pour ces fichiers.
